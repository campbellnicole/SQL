SQL code used for analysis:

select
b.name,
b.city,
c.category,
b.stars,
h.hours,
b.review_count,
b.address,
b.postal_code
from (business as b inner join category as c on b.id = 
c.business_id) inner join hours as h on h.business_id =
b.id
